# spatio-temp-seg-flair1 

> 🚧 **Note:** This repository is currently under construction.

Training and evaluation pipeline in a Pytorch Lightning-based framework using a spatio and temporal fusion segmentation model on the freely available FLAIR1 dataset: 
````
https://ignf.github.io/FLAIR/FLAIR1/flair_1.html
````
<br>

The fusion model architecture is derived by the following paper:
````
https://www.mdpi.com/2072-4292/13/14/2723
````
